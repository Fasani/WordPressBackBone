About my site...

    A work in conistant progress, my personal playground! This where I play with new technologies, the backend is hosted in Word Press for future proofing but this current front end is built using the following technology stack:

    • WordPress
    • WordPress JSON API
    • Gulp for Task Running
    • Require for AMD
    • Backbone for MV*
    • Underscore for Utils
    • jQuery for Awesomeness
    • Moment for Dates
    • Rainbow for Code Highlighting
    • SASS for CSS

    This project can be found on my github!
